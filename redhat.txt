How to create a redhat based LVM mount from Debian:

This needs to be integrated somehow into create_schroot, if possible.

lvcreate -L2G /dev/aquitard/schroot-fedora20-i386
mkfs.ext4 /dev/aquitard/schroot-fedora20-i386
mount /dev/aquitard/schroot-fedora20-i386 /mnt
rpm --root /mnt -ivh /tmp/fedora-release-20-3.noarch.rpm
yum --installroot=/mnt install yum --nogpgcheck
echo i686-redhat-linux > /etc/rpm/platform
yum install zsh vim less rpmbuild yum-utils
